============
Installation
============

At the command line::

    $ pip install django-helpscout

Or, if you have virtualenvwrapper installed::

    $ mkvirtualenv django-helpscout
    $ pip install django-helpscout

Once installed, add ``django_helpscout`` to your ``INSTALLED_APPS``::

    INSTALLED_APPS = (
        ...,
        'django_helpscout'
        ...,
    )
